# EngineerFlow

A scalable task management and analytics platform built with:

- FastAPI (async backend)
- PostgreSQL (Neon)
- MongoDB (Atlas)
- Redis + Celery (background tasks)
- Docker (containerization)

## Features

- JWT authentication
- Project & task management
- Activity logging (MongoDB)
- Async email notifications (Celery)

## Run locally

```bash
docker-compose up --build
